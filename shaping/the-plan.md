# The Plan

This document will describe how to build the product, and track the progress towards building it. 

Phases are numbered and named (e.g. Phase 1: Starting from scratch). Each phase will have a short description of the intended result and how to accomplish it. It will also have a list of goals, formatted as a markdown TODO list, that is updated as the product is built. At the end of each phase section, there will be a short summary of how to verify if the phase is successfully built (through use of the product).

This document is updated as the product is built. Goals are marked done as they are completed, and phase headings are annotated `[COMPLETED]` in their headings when they are finished.
